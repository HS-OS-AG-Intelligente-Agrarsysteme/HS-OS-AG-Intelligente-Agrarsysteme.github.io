# AG Intelligente Agrarsysteme

Website for the Intelligent Agricultural Systems (Intelligente Agrarsysteme) research group at University of Applied Sciences Osnabrück.

## Prerequisites

- Node.js v18.20.8 or v20.3.0, v22.0.0 or higher
- npm

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/HS-OS-AG-Intelligente-Agrarsysteme/HS-OS-AG-Intelligente-Agrarsysteme.github.io.git
cd HS-OS-AG-Intelligente-Agrarsysteme.github.io
npm install
```

## Local Development

Start the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:4321`

## Building

Build the site for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions.
